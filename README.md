## ⚙️ Cómo activar la serpiente animada

> Sigue estos **4 pasos** y la tendrás funcionando en minutos:

**① Crea el archivo del workflow** dentro de tu repo de perfil (`Pipkon/Pipkon`) en esta ruta exacta:
```
.github/workflows/snake.yml
```

**② Pega este contenido** dentro de ese archivo:

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 */12 * * *"   # se ejecuta automáticamente cada 12h
  workflow_dispatch:           # también puedes lanzarlo manualmente

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: Pipkon
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark

      - uses: crazy-max/ghaction-github-pages@v3
        with:
          target_branch: output   # guarda el SVG en la rama "output"
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

**③ Activa permisos de escritura** en tu repo:
`Settings` → `Actions` → `General` → sección **Workflow permissions** → marca **"Read and write permissions"** → Guardar.

**④ Ejecuta el workflow manualmente** la primera vez:
`Actions` (pestaña del repo) → `Generate Snake` → botón **"Run workflow"** → espera ~30 segundos. ¡Listo! 🐍
